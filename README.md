# Loopstudios landing page - Frontend Mentor Challenge

A fully responsive*VR company landing page completed as part of a Frontend Mentor challenge.  
All **HTML**, **CSS**, and **JavaScript** were written by me based on the provided design reference.  
The layout recreates the original design closely while maintaining clean, semantic code and responsive behavior across devices.

## Features

- **Pixel-perfect recreation** of the Frontend Mentor design  
- Fully **responsive layout** for mobile and desktop  
- **Custom CSS** without frameworks  
- **Interactive elements** built with vanilla JavaScript  
- Clean, semantic HTML structure  
- Organized, maintainable file structure  

## Tech Stack

- **HTML5**  
- **CSS3** (Flexbox, Grid, responsive design)  
- **JavaScript (Vanilla)**
